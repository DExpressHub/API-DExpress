## ✨ Funcionalidades da Aplicação

- Cadastro e login com autenticação JWT
- Perfis de usuários com distinção entre famílias e domésticas
- Agendamento de serviços domésticos
- Avaliação e feedback de profissionais
- Gerenciamento de disponibilidade das domésticas
- Integração com banco de dados via Prisma ORM
